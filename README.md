# Multiagent demos (task1 / task2)

Этот репозиторий содержит два простых демонстрационных скрипта на Python, показывающих работу распределённого алгоритма усреднения с использованием SPADE-агентов.

Файлы
- `task1.py` — упрощённый демо-скрипт для нескольких агентов (усреднение). Работа скрипта основана на идее простой децентрализованной пересылки сообщений между агентами.
- `task2.py` — основная демонстрация: агенты обмениваются сообщениями в цикле, после завершения центр собирает траектории и рисует два графика (состояния агентов и их среднее). Мтод усреднения основан на идее локального голосования. Поддерживает шум в данных и другие усложнения.

Требования (рекомендуется виртуальное окружение)
- Python 3.8+
- SPADE (агентная платформа)
- numpy
- matplotlib

Пример установки (fish shell)
```fish
# активировать виртуальное окружение (при наличии)
source .venv/bin/activate.fish
# установить зависимости
pip install spade numpy matplotlib
```

Запуск
```fish
# Запуск первого демо
python task1.py

# Запуск второго демо (отобразит два графика в одном окне)
python task2.py
```

```
- Для воспроизводимости генерации графа и значений добавьте фиксирование сида:
```python
import random
random.seed(42)
```
- Если необходимо сохранить графики вместо прямого вывода, замените `plt.show()` на сохранение файла:
```python
fig.savefig("convergence.png", dpi=150)
```
- Если нужно сохранить результаты выполнения (стоимость, количество сообщений и т. п.), перенаправьте вывод в файл:
```fish
python task2.py > run.log 2>&1
```
